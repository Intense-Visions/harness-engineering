# Discovery: Graph-Aware Context System

## 1. What problem are you solving?

Create the best context system possible for use with AI interaction. This should include context that goes beyond just code — organizational knowledge, telemetry, relationships, intent.

## 2. Hard constraints

- Stay within the TypeScript monorepo
- No other hard constraints

## 3. Soft preferences

- Big-bang approach is acceptable (not incremental)
- OK with heavier dependencies (tree-sitter, vector DBs, etc.)

## 4. What has been considered?

- Take what is good out of OmniContext as inspiration
- No need to integrate directly with OmniContext
- Cherry-pick the best ideas, not the code

## 5. Success in 6 months

- Graph-aware context assembly in Harness
- OmniContext deprecated — Harness becomes the single system
