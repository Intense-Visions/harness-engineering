## What is Harness Engineering?

Harness Engineering is a comprehensive toolkit for transitioning from manual coding to agent-first development. It provides:

- **Core Library**: TypeScript/Python/Go libraries for validation, linting, and agent orchestration
- **CLI Tools**: Command-line tools for scaffolding, validation, and cleanup
- **Agent Skills**: Reusable skills for context engineering, entropy management, and more
- **Documentation**: Complete guides, reference docs, and API documentation

## Quick Start

```bash
# Install the CLI
npm install -g @harness-engineering/cli

# Initialize a new project
harness init my-project

# Validate your harness
harness validate
```
